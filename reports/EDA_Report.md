# Exploratory Data Analysis (EDA) Report

## Dataset Overview

The Titanic dataset contains information about passengers aboard the RMS Titanic, including age, gender, passenger class, fare, and survival status. The objective of this analysis is to explore the dataset and identify factors that influenced passenger survival.

## Data Cleaning

* Loaded the dataset using Pandas.
* Checked for missing values in the dataset.
* Reviewed data types and column information.
* Examined the overall structure and quality of the data.

## Statistical Summary

Descriptive statistics were generated to understand the distribution of numerical variables such as Age and Fare.

Key observations:

* Passenger ages varied across a wide range.
* Ticket fares showed significant variation among passengers.
* The dataset contained both categorical and numerical features.

## Visualizations

### Histogram

A histogram was created to visualize the distribution of numerical variables and understand data spread.

### Correlation Heatmap

A correlation heatmap was generated to identify relationships between numerical features in the dataset.

## Key Findings

1. Passenger class had a noticeable impact on survival rates.
2. Female passengers generally had higher survival rates than male passengers.
3. Higher fare values were often associated with higher passenger classes.
4. Some columns contained missing values that required attention during analysis.

## Conclusion

The exploratory data analysis provided insights into the Titanic dataset and highlighted factors that influenced passenger survival. Passenger class, gender, and fare appeared to be important variables. Visualizations and statistical summaries helped reveal patterns and relationships within the data.
